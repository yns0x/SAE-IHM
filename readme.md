# SAE IHM JavaScript

## Description :
Ce projet est un jeu memory développé en Javascript dans le cadre de la SAE2.02.
Le joueur doit retrouver les paires de cartes le plus  rapidement possible.

## Les fonctionnalités que nous envisageons d'ajouter avant le jour de la recette (Jeudi 21 mai)

- Page de résultats avec récap (temps total et fautes)
- Tableau des scores affiché sur la page d'accueil (le top dans l'API)
- Animation sympa quand il y a une victoire
- Cartes qui tremblent quand c'est une mauvaise paire (permet de bien faire comprendre au joueur directement que le combo n'est pas bon)
- Prévisualisation des cartes de la collection choisie dans le menu (du form)
- Mode contre la montre où le chrono devient un minuteur avant la défaite
- Message personnalisé selon la performance temps et fautes commises ("Champion", "Peut mieux faire", "C'est vraiment le max que tu peux faire là ?")
- mode chaos/ADHD (alerte epilepsie) : Ce mode est fait pour te déconcentrer et challenger ta concentration à son max afin de finir le plus rapidement possible : des objets qui vont dans tout les sens, un gameplay subway surfer fait par nous-même pour le mettre en arrière plan ou sur les côtés du jeu, une sorte de calcul mental rapide qui vient à des moments aléatoires et bloque les events de clic tant que tu ne répond pas
- mode shuffle : toutes les 5/10 secondes, les cartes changent tous de position pour perturber le joueur
- selection de thème qu'on aura fait nous même dans le CSS en haut à droite dans le lobby (celui là est purement esthétique)

## Changements purement esthétique (Quality Of Life)

- changement du thème principal du jeu (moyennement agréable à rester)
- Ajout d'un chronomètre visuel (peut ajouter un peu de stress aux tryharders)
- changement des masques de cartes
